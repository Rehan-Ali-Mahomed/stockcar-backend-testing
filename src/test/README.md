# Backend Testing

### Installation

#### Prerequisites
- npm >= 9.2.0
- node >= 18.13.0

#### Install dependencies and packages

```bash
npm install
```

### Test Execution

#### Prerequisites

Change the variable **API_Path** in the file ```main.test.js``` :
```
http://{ip_address/dns}:{port}/esieaBack/rest
```

#### Launch tests

```bash
npm test
```
